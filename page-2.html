About us

We build a simple, fast website that puts the content first. No clutter, no heavy frameworks — just clean pages that load quickly and read well on any screen.

This project is a hands-on exercise in vanilla HTML and CSS: a shared design system, reusable components, and responsive layouts built entirely with Flexbox.

  • Lightweight

    No libraries or build steps — only the essentials.

  • Responsive

    Layouts adapt from desktop down to mobile.

  • Accessible

    Semantic markup that everyone can navigate.

How simpleWeb started

simpleWeb began as a small experiment: could we build a fast, good-looking website using nothing but the platform itself — plain HTML and CSS? No build tools, no megabytes of JavaScript, no framework churn to keep up with.

The answer turned out to be a confident yes. What started as a class assignment grew into a shared design system, reusable components, and pages that load before you can blink — proof that the fundamentals still go a long way.
